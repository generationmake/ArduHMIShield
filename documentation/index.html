ArduHMI

Graphic HMI for Arduino

ArduHMI with EA DOGM162 on a Arduino Uno.

Introduction

ArduHMI is a HMI with a graphic display and buttons for a Arduino Uno and compatible board.

Hardware

arduhmi_1.1.zip
The hardware is released under CERN OHL v1.2

Software

You can find some sample software on github: https://github.com/generationmake/arduhmi
